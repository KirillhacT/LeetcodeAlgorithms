Как подобрать правую и левую границу (задача на кирпичики)
l чаще всего приравнивается к 0 (или -1) - плохое
r - хорошее
r <= 2 * min(хорошее X)
r = 1, 2, 4, 8, 16... return r
2^k -> плохое, 2^k+1 -> хорошее 
тогда 2^k -> l, а 2^k+1 -> r

задача на котиков: нужно найти мин время, за которое все котики могут собраться
у нас есть массив котиков x[i] и скорости v[i]
t - хорошее, если за t можно собраться

l = 0, за 0 никто собраться не сможет
r : минимальная степень двойки = 10 ^ 10
EPS = 10 ^ -6
while r - l > EPS:
    m = (l + r) // 2
    if good(m):
        r = m
    else:
        l = m
return r
